# 🏠 Airbnb Clone

A full‑stack web application inspired by Airbnb that allows users to browse property listings, view details, create accounts, and manage accommodations. This project replicates core Airbnb functionalities while demonstrating modern full‑stack development practices.

---

## 📌 Project Overview

The **Airbnb Clone** is a property rental platform where users can explore listings, view property information, and interact with hosts. The application focuses on responsive UI design, RESTful backend architecture, authentication, and database management.

This project was developed as a **major academic project** to strengthen full‑stack development skills including frontend development, backend APIs, and database integration.

---

## ✨ Features

### 👤 User Features

* User registration and login authentication
* Browse available property listings
* View detailed property pages
* Responsive UI for mobile and desktop
* Search and explore accommodations

### 🏡 Host Features

* Add new property listings
* Edit and delete listings
* Upload property details and images

### ⚙️ System Features

* RESTful API architecture

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/airbnb-clone.git
cd airbnb-clone
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Setup environment variables

Create a `.env` file in the root directory and add:

```
PORT=3000
MONGO_URI=your_mongodb_connection_string
SESSION_SECRET=your_secret_key
```

### 4️⃣ Run the application

```bash
npm start
```

Open your browser and visit:

```
http://localhost:3000
```
